
### 原因分析

`nvidia-smi` 是**瞬時採樣**工具，可能在以下時刻顯示 0%：

1. **數據載入階段**（CPU 處理）
   - 讀取圖片
   - 解碼
   - 數據增強
   - 轉換為 tensor

2. **Validation 階段**
   - 計算指標
   - 寫入日誌
   - 保存 checkpoint

3. **Epoch 切換**
   - Dataset shuffle
   - DataLoader 重新初始化

### 實際訓練狀態 ✅

**訓練速度**：6-7 批次/秒（持續穩定）
**當前進度**：Epoch 29/150（19.3% 完成）
**GPU 記憶體**：28.6GB / 32GB（89% 使用）
**溫度**：41°C（非常涼爽）
