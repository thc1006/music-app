## 🎓 關鍵洞察（Top 3）

### 1. 短期實驗結果不可泛化 ⚠️
Ultralytics 官方明確警告：30 epochs / fraction=0.1 的最佳參數很少適用於完整訓練。  
→ **啟示**: 當前 20-epoch smoke test 僅用於穩定性驗證，不應據此做長期超參數決策。

### 2. 數據不平衡是主要瓶頸 📊
專案 logs 顯示 duplicate labels + 弱類別（barline_double, tie）問題。  
→ **啟示**: 優先處理順序應為：標註品質 > Oversampling > Loss 權重 > 超參數調整。

### 3. 當前配置過於保守 🚀
`batch=4`, `workers=20`, `amp=false`, `cache=false` 在 RTX 5090 下有大幅優化空間。  
→ **啟示**: 可安全提升至 `batch=8-12`, `workers=8`, `amp=true`, `cache=ram`，預期訓練速度提升 30-50%。
